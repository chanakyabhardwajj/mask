Per-Character Placeholder

playing with React.js

This is a simple demo of a credit-card form built with React.js. Instead of a normal text input element with a placeholder attribute, this demo uses span elements to render each character.
The key-down events are intercepted and processed to fill/delete the characters in every field.

As is obvious, this is a very controlled demonstration. To make this production ready the ancillary features (like copy, paste, select etc.) need to be implemented.

On mobile an invisible input element is used to open the soft-keyboard.
